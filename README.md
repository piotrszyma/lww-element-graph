# LWWElementGraph

An implementation of a state-based LWW graph.

## Installation

```
poetry install
```

## Run tests

```
poetry run pytest
```

## Run coverage (html)
```
poetry run pytest --cov=lww_element_graph tests/
```

## Run coverage (html)
```
poetry run pytest --cov-report html --cov=lww_element_graph tests/
```
